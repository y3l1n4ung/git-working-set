# Git Working Set

A focused workspace for auditing local Git changes.

- **Git Changed Tree:** View all tracked and untracked changes in a clean file hierarchy.
- **Single File Diff:** Open read-only side-by-side comparisons for any file.
- **Multi Diff Review:** One-click to trigger a unified PR-style review of all changes.
- **Summary Changed:** Line-level stats directly in the sidebar title: `Working Set (+12 -3)`.
