# Git Working Set

A focused view for your VS Code Explorer that only shows files with active Git changes. 

## Features

- **Focused View:** Hides all files and folders that are ignored or unchanged, allowing you to focus purely on your current task.
- **Git Integration:** Automatically detects Modified, Untracked, and Staged files using VS Code's built-in Git extension.
- **Hierarchical Tree:** Displays your changes in a familiar tree structure, preserving folder hierarchy for context.
- **Instant Updates:** The view refreshes automatically whenever you save a file or change git status.
- **Visual Feedback:** Uses standard VS Code file icons and Git decorations (colors and badges) for a native feel.

## Requirements

- VS Code 1.74.0 or higher.
- A project initialized as a Git repository.

## Extension Settings

This extension currently has no configurable settings. It works out of the box!
